# biffud.com

## Development

### Run website locally

```
$ yarn site
```

### Run docs locally

```
$ yarn docs
```

### Clear local cache (excludes node_modules, yarn files)

```
$ yarn clear
```

### Clean up (includes node_modules, yarn files)

```
$ yarn clean
```

## Deployment

## Build website

```
$ yarn build:site
```

## Build docs

```
$ yarn build:docs
```
